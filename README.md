# TroškoviApp

## Opis

Mobilna aplikacija za praćenje ličnih troškova i budžeta.

## Tehnologije

- React Native
- Expo
- Expo Router
- Supabase
- TypeScript

## Funkcionalnosti

- Registracija korisnika
- Prijava korisnika
- Dodavanje troškova
- Upravljanje budžetom
- Analitika
- Postignuća
- Profil korisnika
- Notifikacije
- Upload profilne slike

## Pokretanje projekta

npm install

npx expo start
